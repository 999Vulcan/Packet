          После установки Интегратор "Packet"  общается с пользователем
       на русском языке.  Все текстовые строки ( в дальнейшем - фразы )
       при  этом  находятся  в  файле  PackMain.Exe.   В  комплекте   с
       Интегратором могут поставляться специальные файлы с  расширением
       .lng (в дальнейшем  - файлы языка),  в которых содержатся  фразы
       на других языках.
          Чтобы  Интегратор   использовал  такой   файл  языка   вместо
       встроенных в файл  PackMain.Exe  фраз достаточно просто  указать
       его имя с помощью  команды "Опции|Конфигурация|Язык" и  записать
       конфигурацию  с  помощью  команды  "Опции|Записать конфигурацию"
       (если, конечно, Вы не используете автозапись).
          Для  того,  чтобы  вернуться  на  русский  язык,  нажмите   в
       диалоговом окне  ввода имени файла языка  кнопку "Очистить" (по-
       английски - "Clear").
