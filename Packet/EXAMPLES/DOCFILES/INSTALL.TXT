          Интегратор "Packet" поставляется в виде саморазворачивающего-
       ся архива со встроенной программой инсталляции.
          Для инсталляции просто запустите его, выберите тип  инсталля-
       ции (с примерами использования  или без них) и  укажите  целевой
       каталог. Если указанный Вами  каталог  отсутствует  на диске, он
       будет создан при инсталляции. Допускается установка  поверх ста-
       рой версии Интегратора без ее удаления.
          По окончании инсталляции дистрибутивный архив можно удалить с
       жесткого  диска, предварительно  сделав  его резервную копию  на
       дискете.
          Дистрибутивный архив содержит следующие файлы (не считая при-
       меров) :

            1. Read.Me         -  информация по запуску
            2. Packet.Exe      \_ собственно
            3. PackMain.Exe    /  Интегратор
            4. PM386.Exe       -  версия PackMain для DPMI
            5. RTM.Exe         \_ служебные файлы для поддержки работы    
            6. DPMI16BI.Ovl    /  в режиме DPMI
            7. Packet.Hlp      -  ситуационная помощь к Интегратору
            8. Packet.Doc      -  настоящее Руководство
            9. ChkModes.Exe    -  утилита обнаружения SVGA-режимов
           10. Packet.Ico      -  иконка Интегратора для MS Windows
           11. Packet.Pif      -  PIF-файл Интегратора для MS Windows
           12. Russian.Lng     -  файл русского языка
           13. English.Lng     -  файл английского языка
           14. What's.New      -  описание изменений в последних
                                  версиях Интегратора
           15. *.Pal (18 шт.)  -  палитры цветов Интегратора
           16. How2Use.Txt     -  порядок регистрации Интегратора
           17. Overview.Txt    -  краткое описание возможностей
                                  Интегратора
           18. BBS.Txt         -  список BBS, на которых появляются
                                  свежие версии Интегратора "Packet"
                                  
           19. OS2Lnch.Com     -  Эта маленькая (1 K) резидентная
                                  программа позволяет запускать про-
                                  граммы OS/2 из VDM-сессий.
                                  Пользователям OS/2 рекомендуется
                                  включить ее в autoexec.bat.
                                  Благодаря этой программе Вы, в
                                  частности, можете подключать програм-
                                  мы OS/2 в качестве пунктов оглавлений
                                  Интегратора и использовать их в каче-
                                  стве обработчиков по расширению (мас-
                                  ке) имени файла пункта.
                                  Автор - Denis Bychkov, все авторские
                                  права сохранены за разработчиком.

          Полный  комплект  Интегратора  занимает  на винчестере  около
       1.2 M.
          Минимальный   необходимый   для   работы   комплект  -  файлы
       "Packet.Exe",  "PackMain.Exe"  и  "Packet.Hlp" - занимает  около
       400 K.
          В принципе,  возможно использование  Интегратора  и без файла
       помощи "Packet.Hlp",  так  что   если  Вы   готовы  пожертвовать
       ситуационной помощью, то можете  сэкономить еще порядка 120 K на
       Вашем диске.  Автор, правда,  не рекомендует  экономить место на
       диске  таким образом, т. к. работа с Интегратором  будет гораздо
       приятнее и плодотворнее, если Вы будете своевременно заглядывать
       в помощь, а не применять  метод "научного тыка". Имейте в  виду,
       что  некоторые  аспекты  использования  Интегратора, описанные в
       помощи, еще  не отражены  в документации.  Да и  документация не
       всегда и не у всех есть под рукой.
          Обратите внимание  на файл  "What's.New" !  Он содержит самую
       свежую  информацию  об  изменениях  в  Интеграторе.  Часть  этой
       информации еще не включена в ситуационную помощь и документацию.
