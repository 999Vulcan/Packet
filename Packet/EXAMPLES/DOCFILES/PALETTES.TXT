          В комплект  поставки   Интегратора  "Packet"   входят  файлы,
       содержащие  палитры  цветов  Интегратора.   Работа  с  палитрами
       подробно  описана  в п. [    ]:Colors .  Здесь будут лишь  перечислены и
       кратко охарактеризованы имеющиеся цветовые палитры.
          
          Bl&White - палитра по умолчанию для черно-белых мониторов
          Blue     - голубая палитра
          Brown    - коричневая палитра для LCD-мониторов
          Default  - стандартная палитра, принимаемая  по умолчанию для
                     цветных мониторов
        ! Green    - зеленая палитра; оптимальный выбор  для длительной
                     работы, т. к. не давит на глаза
          Grey     - серая палитра
          Grey&Blu - серая палитра с синими окнами оглавлений
        ! Grey_New - модернизированная серая палитра; наряду  с зеленой
                     очень хороша для длительной работы
          GSU      - палитра ГГУ; сделана наспех, но некоторым нравится
          LightBlu - светло-голубая палитра
          Mono     - палитра по умолчанию для монохромных мониторов
          Norton   - палитра, похожая на Norton Utility
          Red      - красная палитра для LCD-мониторов
          Retro    - палитра "под старину"
          Retro`   - палитра "под старину" с другим цветом обоев
          Strict   - строгая палитра, не давящая на глаза
          Strict`  - строгая палитра с другим цветом обоев
        ! White    - очень  красивая  палитра  с белыми  меню  и окнами
                     диалогов
