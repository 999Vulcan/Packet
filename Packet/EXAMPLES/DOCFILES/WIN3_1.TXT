          Если   Вы   запускаете   Интегратор  "Packet"   в  DOS-сессии
       Windows 3.1, Вы можете запускать любые программы Windows 3.1.
          Правда, в отличие  от двух  предыдущих  случаев, процесс этот
       несколько более сложный.

          Для начала Вам потребуется  произвести инсталляцию  программы
       WXServer. Опишем этот процесс по шагам:

          1. Скопируйте файл  VMB.386 из каталога Интегратора в каталог
       Windows.

          2. Добавьте в секцию [386Enh] файла "system.ini" строчку

          device=vmb.386

          3. Запустите  Windows  и включите  в Группу Запуска программу
       WXSRVR.EXE из каталога Интегратора.

          4. Перезапустите Windows.

          В  заключение  добавим, что в ближайших  версиях  Интегратора
       процесс инсталляции программы WXServer будет автоматизирован.

          Теперь  программа WXServer готова к использованию и Вы можете
       запускать  программы  Windows 3.1  из сеанса DOS, как  с помощью
       Интегратора "Packet",  так  и  без  него.  Для запуска программы
       Windows 3.1  необходимо  запустить в DOS-сессии программу WX.EXE
       из  каталога  Интегратора  и передать ей  в качестве  параметров
       строку "/a <имя_программы> <параметры_программы>".
          Эту технологию  Вы  можете  использовать как  для подключения
       программ Windows 3.1 как пунктов, так и для использования их как
       обработчиков по расширению или маске имени файла.
