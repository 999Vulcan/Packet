          Как  уже  было  сказано,  Интегратор  -  многооконная  среда,
       поэтому   текстовый   редактор   Интегратора   также    является
       многооконным.
          Открытие   окна   редактора  с  существующим  на диске файлом
       осуществляется по команде "Файлы|Открыть" (F3).
          Если  в   появившемся  диалоговом   окне  Вы   выберете  файл
       оглавления,  то   откроется  окно   оглавления,  иначе   -  окно
       текстового редактора. Вы можете ввести несуществующее имя файла,
       чтобы создать новый текстовый файл.
          Открыть   пустое    окно      текстового    редактора   можно
       также  по  команде "Файлы|Новый файл" (Shift+F4).
          Внизу подменю  "Файлы" отображается список 5 последних закры-
       тых текстовых файлов. Вы можете выбрать любой из них для загруз-
       ки  во встроенный редактор. Эта возможность доступна  только для
       зарегистрированных пользователей.
          Запись  текстового   файла  из   активного  окна    редактора
       осуществляется с помощью команды "Файлы|Записать" (F2). Если  Вы
       хотите записать файл под другим именем, воспользуйтесь  командой
       "Файлы|Записать как".  Для  записи   всех  измененных  текстовых
       файлов воспользуйтесь командой "Файлы|Записать все".
          Все эти команды доступны также из локального меню редактора.
