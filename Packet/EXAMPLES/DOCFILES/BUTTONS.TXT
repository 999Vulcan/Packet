          Нажатие  на   любую  такую   кнопку  приводит   к  выполнению
       определенных действий.
          Так,  кнопка  "ОК"  зарезервирована  для  установки внесенных
       изменений и перехода к выполнению команды.
          Кнопка  "Отмена"  предназначена  для  завершения  работы окна
       диалога с игнорированием всех изменений.
          Нажатие кнопки  "Помощь" открывает  окно помощи  для текущего
       окна диалога.
          Перечисленные  поля  выполняют  эти  стандартные действия для
       всех окон диалогов Интегратора.
          Некоторые диалоговые окна имеют также специфичные только  для
       них кнопки.
          Для  нажатия  на  кнопку  необходимо  щелкнуть на этой кнопке
       мышью, или  нажать на  клавиатуре клавишу  с выделенной  литерой
       кнопки (можно в  сочетании с клавишей  Alt), или сделать  кнопку
       активной с помощью клавиш  Tab (вперед) или Shift+Tab  (назад) и
       нажать Enter.
          Если  ни  одна  из  кнопок  в  данный  момент  не активна, то
       нажатие клавиши Enter повлечет нажатие так называемой кнопки  по
       умолчанию, которая выделена особым цветом.
