          Каждому   расширению   или   маске   в   списке  обработчиков
       отводится  отдельный   блок,  начинающийся   с  символа   '>'  и
       следующей за ним без пробела маски имени файла.
          Далее  в  той  же  строке  через  один или несколько пробелов
       можно  указать  имя  программы-обработчика  или  команды MS DOS,
       являющейся обработчиком  по умолчанию  (т. е.  по клавише Enter)
       для файлов с именем, удовлетворяющим маске.  Имя обрабатываемого
       файла подставляется в конец строки автоматически при его выборе,
       если имя обработчика  не содержит  ни одного из следующих макро-
       сов:

          !   - имя обрабатываемого файла
          !.! - имя с расширением обрабатываемого файла
          !\  - каталог обрабатываемого файла (без завершающего символа
                '\')
          !!  - восклицательный знак

          Если обнаруживается один из этих макросов, выполняется макро-
       подстановка, а имя файла в конец строки не подставляется.
          Если обработчик по умолчанию не указан, файл будет  грузиться
       во встроенный текстовый редактор.
          Если Вы хотите, чтобы  по клавише Ctrl+Enter вместо  загрузки
       во  внешний  текстовый  редактор  происходил  вызов   какой-либо
       другой  программы,  следом  должна  идти  строка, начинающаяся с
       символа '#'.  После этого символа (можно без пробела) необходимо
       указать имя программы-обработчика или команду MS DOS.
          Далее до  следующей строки,  начинающейся с  '>', идет список
       обработчиков, который  будет предлагаться  для файлов  с именем,
       удовлетворяющим маске.
          Каждому  обработчику  должна   соответствовать  одна   строка
       в  списке,  причем  часть  этой  строки  до  символа  '^'  будет
       являться  собственно  именем  обработчика,  а  остальная  -  его
       названием,  которое  будет  выводиться  в  списке  при   нажатии
       Shift+Enter.
          Если в  строке отсутствует  символ '^',  то эта  строка будет
       считаться и названием, и собственно именем обработчика.
          При желании  Вы можете  выделить по  одной букве  в названиях
       символами  '~'  с  двух  сторон.  Тогда  при выборе из списка Вы
       сможете использовать горячие клавиши.
          Строка,  содержащая  только  символ  '-',  вставляет в список
       разделитель.
          Пустые строки и строки, начинающиеся с ';', игнорируются.
          Такой  же  список  может  находиться  в самом начале описания
       обработчиков. Это  общий список.  Он будет  добавляться в начало
       списка, выдаваемого по Shift+Enter, для каждого типа файлов.
