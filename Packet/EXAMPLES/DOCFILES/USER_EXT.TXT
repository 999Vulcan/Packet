          Интегратор  позволяет  Вам  указать,  какие   действия  будут
       выполняться  при   выборе  пункта   оглавления  Интегратора    в
       зависимости  от  расширения   или маски  имени  соответствующего
       файла, реакцию на Ctrl+Enter, а также определить набор действий,
       который   будет   предлагаться  при   выборе  пункта   клавишами
       Shift+Enter.
          Для    этого    существует    две    команды    в     подменю
       "Оглавления|Обработка" :  "Локальные обработчики"  и "Глобальные
       обработчики".
          Глобальный  список  хранится  в  файле  "Packet.Ext" в том же
       каталоге, в  котором инсталлирован  Интегратор, и  действует для
       тех  оглавлений,  для  которых  не  создан свой список с помощью
       команды "Локальные обработчики".
          Локальный  список  хранится  в  файле  оглавления и действует
       только для того оглавления, для которого создан.
          Если Вы используете сразу много оглавлений, Вам будет удобнее
       использовать глобальный  список, создаваемый  с помощью  команды
       "Глобальные обработчики".
