          Работа  с  блоками  в  редакторе Интегратора происходит через
       "Карман" (русский  аналог слова  Clipboard; в  дальнейшем -  без
       кавычек).
          Например,  для  копирования  блока  Вам  необходимо   сначала
       выделить блок  с помощью  комбинации клавиши  Shift с  клавишами
       управления курсором, затем скопировать  его в Карман (Ctrl+Ins),
       а  после этого  установить  курсор  в   позицию,  в  которую  Вы
       хотите  копировать  блок.  При  этом   отметка  блока  на экране
       исчезнет,  но  его  копия  останется  в  Кармане. Нажатие клавиш
       Shift+Ins  приведет  к  копированию  блока  из Кармана в позицию
       курсора.
          Действия  при  переносе  блока  аналогичны  описанным   выше,
       только вместо  копирования блока в  Карман  необходимо выполнять
       перенос.

.db -

                ║ Ctrl+Y        ║ Удалить текущую строку      ║
.de
                ║ Shift+стрелки ║ Отметить блок текста        ║
                ║ Shift+Del     ║ Перенести блок в Карман     ║
                ║ Ctrl+Ins      ║ Скопировать блок в Карман   ║
                ║ Shift+Ins     ║ Скопировать блок из Кармана ║
                ║ Ctrl+Del      ║ Удалить отмеченный блок     ║
                ╚═══════════════╩═════════════════════════════╝

          Последние  четыре  команды  можно  также  выбрать  в  подменю
       "Редактор".
          Содержимое Кармана можно  просмотреть, используя команду меню
       "Редактор|Показать Карман".
          Используя  окно  Кармана  и команды работы с  блоками,  можно
       организовать запись  блока в  файл  и чтение блока из  файла   в
       позицию курсора.
          Для записи блока в файл необходимо скопировать его в  Карман,
       а  затем,   открыв  окно   Кармана,  воспользоваться    командой
       "Файлы|Записать как".
          Для чтения  блока из  файла в  позицию курсора  можно открыть
       окно  редактора  с  этим  файлом,  скопировать  его содержимое в
       Карман   (Ctrl+Ins),    а   затем    воспользоваться    командой
       "Редактор|Приклеить" (Shift+Ins).
